JOYFIT24経堂 5月スタジオイベント LP — セットアップ手順
=====================================================

■ 同梱物
- index.html … 静的LP＋申込フォーム（Tailwind CDN）
- gas/Code.gs … スプレッドシート追記用Google Apps Script

■ スプレッドシート
URL（スプレッドシートIDはコード内にも記載）:
https://docs.google.com/spreadsheets/d/1Eba3Uvn4lRK5z4hshHdsOav6yMrpvQov6bT_5TWINag/edit

gid=799051187 のタブ名をブラウザ上で確認し、gas/Code.gs の SHEET_NAME を同じ名前に変更してください。

■ シート1行目（推奨ヘッダー順）
タイムスタンプ, 申込ID, フォーム種別, 店舗名, 氏名, メールアドレス, 電話番号, 参加希望レッスン, 参加希望日, 希望開始時間, 参加希望日時まとめ, 希望内容（自由記入）, 備考, 同意, 申請完了メール送信日時, 管理者通知送信日時, リマインド送信状態, リマインド送信日時

※LPは氏名・メール・電話・参加希望（リスト選択または自由記入）・備考・同意を取得します。
※GAS最新版では「申込ID」「通知送信ログ」「リマインド送信ログ」も保存します。

既に別形式の列がある場合:
- 既存1行目に合わせて gas/Code.gs の HEADER_ROW と appendRow の配列順を編集する
- または空のシート／新規タブを作成して上記ヘッダーを用意する

■ GASデプロイ
1. スプレッドシートを開く
2. メニュー「拡張機能」>「Apps Script」
3. Code.gs の内容を貼り付け（SPREADSHEET_ID / SHEET_NAME を確認）
4. 「デプロイ」>「新しいデプロイ」> 種類「ウェブアプリ」
   - 次のユーザーとして実行: 自分
   - アクセスできるユーザー: 全員（テスト後に制限しても可）
5. 発行されたURLを index.html 内の定数 webAppUrl に貼り付け

【重要】「jf-kyoudou@okamoto-group.co.jp から送信」にする場合
- Apps Script 実行アカウントに Gmail 送信エイリアス設定が必要です。
- gas/Code.gs の USE_GMAIL_ALIAS_FROM を true にして運用します。
- エイリアス未設定時は false のまま運用してください（replyTo は jf-kyoudou になります）。

■ リマインドメール設定（前日送信）
1. Apps Script エディタで関数 setupReminderTrigger を1回実行
2. 承認ダイアログで許可
3. これで毎日18:00（スクリプトTZ）に sendReminderBatch が実行されます
4. 試験送信は sendReminderBatch を手動実行

■ 文言・スケジュールの更新
- index.html 内の EVENT_CONFIG（キャッチ・期間・注意事項）と window.GW_SCHEDULE を、掲示物の内容に合わせて更新してください。

■ ホスティング
- 任意のHTTPSホストに index.html を配置（ローカルfile://ではCORSや挙動が不安定な場合があります）
